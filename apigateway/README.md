# API Gateway

The API Gateway is the principle access service for the client. The API gateway ensures a request is authenticated and authorized before the requested resource is accessed.

```
.

│   ├── main-config.js

├── routes
│   └── apigateway.js
├── app.js
├── package.json
├── package-lock.json
├── README.md
└── server.js
```
